## Demonstração

- **Layout elegante** paleta de cores: (preto, dourado e tons neutros)
- **Totalmente responsivo** - adapta-se perfeitamente a desktop, tablet e dispositivos móveis
- **Menu interativo** com categorias e preços
- **Galeria de imagens** com efeitos hover suaves
- **Sistema de reservas** com formulário funcional
- **Depoimentos** de clientes em cards estilizados
- **Mapa integrado** com localização via Google Maps
- **Navegação suave** entre seções (scroll behavior)
- **Header fixo com efeito glassmorphism** e menu mobile hambúrguer

## Tecnologias Utilizadas
 
 HTML5 - Estrutura semântica e acessível
 CSS3 - Animações - Flexbox - Grid - Variáveis - Glassmorphism
 JavaScript (Vanilla) - Menu mobile - Scroll suave - Formulário - Efeitos header
 Google Fonts - Cormorant Garamond + Montserrat
 Font Awesome 6 - Ícones vetoriais
 Pexels - Imagens do hero - Galeria - About
